# RBX Instance

Roblox의 다중 실행을 가능하게 하는 세션 언락커입니다.

## 기능

- 여러 개의 Roblox 클라이언트 동시 실행 가능
- 활성화 종료 버튼 추가


**참고**: 프로그램은 관리자 권한이 필요합니다. 실행 시 UAC 프롬프트가 나타나면 "예"를 클릭하세요.

## 사용 방법

1. 프로그램 실행
2. 주의사항 팝업 확인
3. "활성화 시작" 버튼 클릭
4. 활성화 완료 후 Roblox를 여러 번 실행
5. 종료하려면 "활성화 종료" 버튼 클릭


## 주의사항

⚠️ **중요**:
- 이 프로그램의 사용으로 인해 발생하는 모든 문제(계정 정지 등)에 대한 책임은 사용자 본인에게 있습니다
- Roblox 이용약관 위반 가능성이 있습니다
- 테스트 계정으로만 사용하세요

## 시스템 요구사항

- Windows 10/11
- .NET 8.0 Runtime
- 관리자 권한 (필수)

## 작동 원리

이 프로그램은 Roblox가 사용하는 `ROBLOX_singletonMutex` 뮤텍스를 선점하여 다중 실행을 가능하게 합니다.
